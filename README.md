UTM Networking Lab

Questa repository contiene un esercizio base di networking realizzato con macchine virtuali in UTM.
Il laboratorio è stato creato per comprendere il funzionamento delle reti virtuali, la
configurazione degli indirizzi IP, del gateway e il test di connettività tra le macchine.

Macchine Virtuali:
   
⁠Kali Linux
⁠Metasploitable
Windows 10

Configurazione di Rete:
   
⁠Modalità di rete: NAT Network (UTM)
Configurazione IP statica e dinamica
⁠Test di connettività eseguiti con il comando ping

Obiettivo
L’obiettivo di questo esercizio è praticare i concetti base del networking in un ambiente
virtuale sicuro e isolato.
